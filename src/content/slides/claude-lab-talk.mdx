## Code Example

```python
def analyze_sequence(dna: str) -> dict:
    """Analyze DNA sequence composition"""
    counts = {'A': 0, 'T': 0, 'G': 0, 'C': 0}

    for base in dna.upper():
        if base in counts:
            counts[base] += 1

    return counts

# Usage
result = analyze_sequence("ATCGATCG")
```
